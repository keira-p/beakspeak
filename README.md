**BeakSpeak**

BeakSpeak is an end-to-end image classification project that identifies bird species from photos using deep learning.

It uses the CUB-200 dataset, compares a baseline CNN with transfer learning models, tracks experiments in MLflow, evaluates performance including confusion patterns, and deploys a simple user-facing app for live inference.

The app allows users to upload a bird image and receive the top predicted species with confidence scores and optional species facts.
